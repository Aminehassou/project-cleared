How does it work?

There's 3 types of clear statuses for games in your backlog. Modifying these statuses
will help you keep track of which games you're currently playing and which games
you've completed.

Playing

The game has not been cleared yet and is still being played by the user

Cleared

The game has been cleared by the user (does not include optional content)

100% Cleared

The game has been fully cleared by the user (includes all optional content)
